# Telegram Reminder Bot (Go)

Простой Telegram-бот напоминалка на Go.

## Что умеет
- `/in <duration> <text>`: напомнить через промежуток времени
- `/at <YYYY-MM-DD HH:MM> <text>`: напомнить в точное время (локальная timezone сервера)
- `/list`: список активных напоминаний
- `/delete <id>`: удалить напоминание

Данные сохраняются в `reminders.json`.

## Быстрый старт
1. Создай бота через `@BotFather` и получи токен.
2. В корне проекта создай `.env`:

```bash
TELEGRAM_BOT_TOKEN=твой_токен
```

3. Установи зависимости:

```bash
go mod tidy
```

4. Запусти бота:

```bash
set -a
source .env
set +a
go run .
```

## Примеры команд в Telegram
- `/in 10m Проверить духовку`
- `/at 2026-03-06 21:30 Выпить воду`
- `/list`
- `/delete 1`

## Важно
- Если приложение остановить, напоминания не отправятся, пока бот снова не запущен.
- Для 24/7 нужен постоянно работающий сервер/VPS/облако.
